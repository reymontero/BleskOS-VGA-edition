# BleskOS-VGA-edition
Part of BleskOS operating system project for oldest computers

## Version Alpha 1
This version contains drivers:
* VGA
* PC speaker
* Keyboard
* PS/2 mouse
* Easy filesystem for USB and hard disk

And contain this programs:
* Text editor
* Graphic editor
